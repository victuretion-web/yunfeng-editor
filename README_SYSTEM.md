# 云锋智能视频生成系统 - 用户手册与最佳实践

本系统是一套集成了 AI 语音识别、语义打点、大模型素材匹配和多轨道视频合成的专业级智能视频生产工作流。

## 一、 系统功能与特性

1. **多轨道智能合成引擎**
   - 支持独立配置口播、产品、病症、音效、贴图等多路素材路径。
   - 自动生成剪映专业版草稿，包含多轨道混剪、画中画效果及贴纸。
2. **AI 语义分析与大模型匹配**
   - 依托 Whisper 模型自动识别口播内容，将其转化为带时间戳的字幕。
   - 内置基于关键词的本地语义分析引擎，自动识别“病症描述”、“产品推荐”等转折点。
   - 提供 **大模型精准中插** 模块，支持接入 `gemini-3.1-pro-preview` 等前沿模型，实现复杂文案的高级匹配。
3. **高并发任务队列**
   - 界面内置了基于 `ThreadPoolExecutor` 的任务队列，最高支持 **5 个并发生成任务**。
   - 一分钟的口播素材平均在 10 秒内即可完成剪映草稿的生成（符合处理速度小于5分钟的要求）。
4. **高质量与稳定性**
   - 采用生成剪映草稿 + 手动二次确认导出的方案，彻底规避了传统 RPA 导出的不稳定与黑屏问题。

## 二、 快速上手指南

### 1. 准备素材
请按照以下目录结构最佳实践来组织您的素材（系统也支持任意位置的散装文件夹，但统一管理更高效）：

```text
D:\我的视频项目\
├── 口播\    (存放带有语音的 MP4 或 WAV 文件)
├── 产品\    (存放产品展示、空镜等 B-Roll 素材)
├── 病症\    (存放痛点描述、负面情绪等素材)
├── 音效\    (存放背景音乐、转场音效等，可选)
└── 贴图\    (存放透明 PNG 或特殊贴纸，可选)
```

### 2. 使用“一键智能合成”工作流
1. 运行 `python ui_main.py` 打开系统。
2. 在【🎬 一键智能合成】标签页中，您可以：
   - 逐个点击“选择文件夹”填入对应的素材路径。
   - **推荐**：在“统一总目录”中填入 `D:\我的视频项目\`，然后点击 **“⬇️ 自动从总目录推导子路径”**，系统会自动补全上面的路径。
3. 选择灵敏度（推荐 `medium`）和输出画质（1080p/4k）。
4. 点击 **“🚀 验证路径并提交生成任务”**。系统会检查必填路径是否合法，验证通过后会自动加入任务队列。

### 3. 查看任务进度
点击【📊 任务队列看板】标签页，您可以实时看到所有任务的排队情况、当前状态和执行日志。

## 三、 素材管理最佳实践 (Best Practices)

为了保证 **内容识别准确率≥90%，素材匹配相关度≥85%**，请遵循以下规范：

1. **口播素材 (Speech)**
   - 建议在安静环境下录制，确保人声清晰，背景噪音小。
   - 若视频本身无画面仅有声音，建议保存为音频格式或使用一张静态背景的 MP4。
2. **中插素材 (B-Roll: Product / Symptom)**
   - 每个素材的长度建议在 **3秒 到 8秒** 之间。太短会导致闪烁，太长会使视频节奏拖沓。
   - 文件名建议具有描述性（例如 `头痛表现.mp4`），这对大模型高级匹配模块非常有帮助。
3. **参数调节 (Sensitivity)**
   - `low` (严肃/低密)：素材间隔长，适合科普类、教学类视频。
   - `medium` (推荐/中密)：适合普通的电商带货、产品推荐。
   - `high` (快节奏/高密)：适合短视频平台的快速吸睛内容。

## 四、 常见问题 (FAQ)

- **问：生成成功了，但在剪映里没看到草稿？**
  答：剪映首页可能不会自动刷新。请点击界面上的“📂 打开草稿目录预览”按钮，双击新生成的工程文件即可强行唤起剪映。
- **问：如何确保 1080p/4K 质量？**
  答：本系统搭建的时间轴默认适配您放入的第一段口播素材的分辨率。您在剪映点击“导出”时，可以自由选择 1080p、4K 甚至是 60fps 码率。
