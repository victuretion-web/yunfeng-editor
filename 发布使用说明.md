# 云锋剪辑绿色版发布说明

## 1. 交付内容

- 发布目录: `dist/YunFengEditor/`
- 主程序: `dist/YunFengEditor/YunFengEditor.exe`
- 校验报告: `dist/YunFengEditor/release_verification.json`
- 运行输出目录: 程序首次启动后会在可执行文件同级自动创建 `output/`

本次发布采用 Windows `PyInstaller one-folder` 绿色版方案。
优点是资源路径稳定，便于携带 FFmpeg、Whisper 模型、剪映 skill 脚本和 Python 运行时。

## 2. 系统要求

- 操作系统: Windows 10 或 Windows 11 64 位
- Python: 不需要额外安装
- 剪映: 专业版 `5.9.x` 或更低兼容版本
- 磁盘空间:
  - 解压后建议至少预留 `6 GB`
  - 运行期建议额外预留 `3 GB` 以上用于输出和缓存
- 内存:
  - 最低 `8 GB`
  - 建议 `16 GB` 或更高

## 3. 安装步骤

1. 将 `YunFengEditor-portable-win64.zip` 解压到本地磁盘目录。
2. 确保解压后的目录结构中包含:
   - `YunFengEditor.exe`
   - `_internal/`
   - `release_verification.json`
   - `发布使用说明.md`
3. 双击运行 `YunFengEditor.exe`。
4. 首次启动后，程序会在同级目录创建:
   - `output/`
   - `.whisper_cache/`（如运行期需要额外缓存）

## 4. 使用步骤

1. 在主界面填写素材目录:
   - `口播素材路径`
   - `产品素材路径`
   - `病症素材路径`
2. 按需填写可选目录:
   - `音效`
   - `背景音乐`
   - `广审`
   - `贴图`
3. 检查大模型参数:
   - `LLM_API_KEY`
   - `LLM_BASE_URL`
   - `LLM_MODEL`
4. 点击批量提交或单次生成。
5. 在任务看板查看执行状态和日志。

## 5. 已内置资源

## 8. 常见问题

### 8.1 双击没有反应

- 确认已完整解压，不要直接在压缩包内运行
- 确认 `YunFengEditor.exe` 同级存在 `_internal/`
- 查看 `output/internal_maintenance.log`

### 8.2 生成失败或任务报错

- 打开 `output/task_logs/`
- 查看对应任务日志
- 检查素材路径是否存在
- 检查口播目录中是否混入音频文件

### 8.3 看不到新草稿

- 重启剪映
- 确认剪映版本为 `5.9.x`
- 检查剪映草稿目录权限是否正常

### 8.4 提示找不到素材或路径错误

- 不要移动 `_internal/` 内的文件
- 不要只拷贝 `YunFengEditor.exe` 单文件
- 确认素材目录为本机真实路径

### 8.5 启动后被杀毒软件拦截

- 将整个发布目录加入信任
- 使用完整解压目录运行，不要拆散文件

## 9. 维护建议

- 交付给同事时，建议直接发送整个压缩包
- 不建议同事手动修改 `_internal/` 内文件
- 如需重新打包，请执行 `build_release.ps1`
