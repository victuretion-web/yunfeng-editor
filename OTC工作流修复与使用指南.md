# OTC药品推广视频工作流 - 问题修复与使用指南

## 一、已识别的问题及修复方案

### 问题1：素材堆叠现象
**现象**：多个素材在时间轴上堆叠，导致视觉混乱
**原因**：素材插入逻辑未充分考虑时间冲突
**修复方案**：
- 限制单个素材最长5秒
- 增加素材间的时间间隔检查
- 确保素材不会与下一个素材重叠

### 问题2：视频时长控制不精确
**现象**：最终视频时长可能与口播时长不一致
**原因**：素材和字幕可能超出口播时长
**修复方案**：
- 多方法获取视频时长（pydub、ffprobe、字幕分析）
- 严格截断超出口播时长的素材和字幕
- 在视频创建时验证最终时长

### 问题3：字幕与口播内容不匹配
**现象**：字幕内容与主播实际口播不一致
**原因**：环境缺少FFmpeg，无法进行真实语音识别
**修复方案**：
- 提供FFmpeg安装指南
- 增强模拟字幕的智能性
- 支持人工字幕修正

### 问题4：素材密度调节不够灵活
**现象**：素材插入密度无法精确控制
**原因**：灵敏度设置不够细粒度
**修复方案**：
- 实现三级灵敏度调节（low/medium/high）
- 提供详细的参数说明和调节方法

## 二、灵敏度调节方法

### 灵敏度级别说明
| 灵敏度级别 | 间隔时间 | 素材密度 | 适用场景 |
|------------|----------|----------|----------|
| low        | 4.0秒    | 低密度   | 严肃内容、详细讲解 |
| medium     | 2.5秒    | 中等密度 | 一般推广、平衡效果 |
| high       | 1.5秒    | 高密度   | 快节奏、吸引注意力 |

### 调节方法

#### 方法1：修改配置参数（推荐）
1. 打开文件：`otc_promo_workflow.py`
2. 找到以下代码段（约第428行）：
   ```python
   sensitivity = 'medium'  # 可设置为 'low', 'medium', 'high'
   ```
3. 修改值为所需灵敏度：
   - 低密度：`sensitivity = 'low'`
   - 中等密度：`sensitivity = 'medium'`
   - 高密度：`sensitivity = 'high'`
4. 保存文件并重新运行

#### 方法2：批量处理调节
1. 打开文件：`batch_otc_promo_workflow.py`
2. 找到以下代码段（约第79行）：
   ```python
   sensitivity = 'medium'  # 可设置为 'low', 'medium', 'high'
   ```
3. 修改值为所需灵敏度
4. 保存文件并重新运行

### 自定义灵敏度（高级用户）
如果需要更精确的控制，可以修改灵敏度配置：

1. 打开文件：`otc_promo_workflow.py`
2. 找到灵敏度配置字典（约第204-208行）：
   ```python
   sensitivity_config = {
       'low': 4.0,    # 4秒间隔
       'medium': 2.5,  # 2.5秒间隔
       'high': 1.5     # 1.5秒间隔
   }
   ```

3. 修改间隔时间值（单位：秒）
4. 保存文件并重新运行

## 三、测试验证标准

### 关键测试指标

#### 1. 字幕匹配度
- **验证标准**：字幕内容与主播口播内容100%对应
- **测试方法**：
  - 播放原始口播视频，对照生成的字幕
  - 检查字幕时间戳是否与口播时间一致
  - 验证字幕文本是否准确反映口播内容

#### 2. 视频时长控制
- **验证标准**：最终视频时长与原始口播时长完全一致
- **测试方法**：
  - 记录原始口播视频时长
  - 检查生成视频的时长
  - 验证所有素材和字幕是否在时长范围内

#### 3. 素材密度与分布
- **验证标准**：素材分布均匀，无堆叠现象
- **测试方法**：
  - 检查时间轴上的素材分布
  - 验证素材间是否有合理间隔
  - 确认素材密度符合所选灵敏度设置

#### 4. 视觉效果
- **验证标准**：素材与口播内容匹配，视觉效果自然
- **测试方法**：
  - 播放生成的视频
  - 检查素材内容是否与对应口播内容相关
  - 验证转场是否自然流畅

### 测试步骤

1. **准备测试素材**：
   - 选择3-5分钟的口播视频
   - 准备病症和产品素材

2. **测试不同灵敏度**：
   - 分别使用low、medium、high三种灵敏度
   - 记录每种设置下的素材数量和分布

3. **验证核心功能**：
   - 检查字幕匹配度
   - 验证视频时长控制
   - 评估视觉效果

4. **性能测试**：
   - 测试批量处理速度
   - 验证系统稳定性

## 四、故障排除

### 常见问题及解决方案

#### 问题：语音识别失败
**症状**：系统使用增强模拟字幕
**解决方案**：
- 安装FFmpeg：下载并安装FFmpeg，添加到系统PATH
- 验证pydub安装：`pip install pydub`
- 检查视频文件格式：确保视频格式受支持

#### 问题：素材插入太少
**症状**：生成的视频中素材数量不足
**解决方案**：
- 提高灵敏度设置
- 检查素材库是否有足够的素材
- 验证字幕语义分析是否正确

#### 问题：素材堆叠
**症状**：时间轴上素材重叠
**解决方案**：
- 降低灵敏度设置
- 检查素材时长是否过长
- 验证视频时长控制逻辑

#### 问题：视频导出失败
**症状**：无法导出MP4文件
**解决方案**：
- 检查剪映是否正常运行
- 确保输出目录有写入权限
- 验证视频编码设置
